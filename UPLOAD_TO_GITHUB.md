# GitHub Pages 上傳方式

把本資料夾內的所有檔案與資料夾上傳到 GitHub repository 根目錄。

正確上傳後，GitHub 根目錄應看到：
- index.html
- app.js
- styles.css
- service-worker.js
- manifest.json
- data/recalls.json
- assets/icon.svg
- .nojekyll

不要只上傳 ZIP，也不要讓檔案被包在 github_pages_upload_ready 或 food_safety_scanner_app 資料夾裡。

測試資料是否可讀：
https://你的帳號.github.io/你的repo名稱/data/recalls.json
